public interface Deque<E> extends Queue<E>

Data Structure : Double ended Queue & Stack
Below is the list of extra methods (not present in Queue Interface) in Deque Interface

Methods for adding element at both end :
boolean offerFirst(E e)
boolean offerLast(E e)
void addFirst(E e)
void addLast(E e)
void push(E e) - Pushes an element onto the stack represented by this deque. Throws IllegalStateException if the element cannot be added due to capacity restrictions.

Methods for getting element from both end :
E peekFirst()
E peekLast()
E getFirst() - Retrieves, but does not remove, the first element of this deque. Throws an exception if this deque is empty.
E getLast() - Retrieves, but does not remove, the last element of this deque. Throws an exception if this deque is empty.

Methods for removing element from both end :
E pollFirst()
E pollLast()
E removeFirst()
E removeLast()
boolean removeFirstOccurrence(Object o) - Removes the first occurrence of the specified element from this deque. If the deque does not contain the element, it is unchanged. Returns true if this deque changed as a result of the call.
boolean removeLastOccurrence(Object o)
E pop() - Pops an element from the stack represented by this deque. Throws NoSuchElementException if this deque is empty
